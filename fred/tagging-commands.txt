First release (create v1.0.0 and v1)

git clone https://github.com/siahmurph/devops-templates.git
cd devops-templates
git checkout main
git pull

# create immutable release tag
git tag -a v1.0.0 -m "Release v1.0.0"

# create/move major channel tag
git tag -a v1 -m "Major v1"

# push both tags
git push origin v1.0.0
git push origin v1


Update later (example: v1.1.0, move v1 forward)

git checkout main
git pull

git tag -a v1.1.0 -m "Release v1.1.0"
git push origin v1.1.0

# move v1 to latest v1 release
git tag -fa v1 -m "Move v1 to v1.1.0"
git push origin v1 --force



Consumers use
uses: siahmurph/devops-templates/.github/workflows/standard-build.yml@v1
